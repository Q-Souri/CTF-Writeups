# Pwn.Careers Capture The Flag Portfolio

This directory contains my technical write-ups for five Capture The Flag (CTF) challenges completed on the **Pwn.Careers** platform.

Each write-up documents my methodology, technical approach, exploitation process, and the security lessons learned while solving real-world inspired web security challenges.
